# Persist

Persist is a simple gem that allows you to painlessly save Ruby objects in a NoSQL persistent store.

## Installation

Add this line to your application's Gemfile:

    gem 'persist'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install persist

## Usage

```db = Store.new```
